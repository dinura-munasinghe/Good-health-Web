Splash Page

Technical Description

The splash page (splash.html) serves as an entry point to the website with a 4-second automatic redirect to the main page. Implemented using the <meta http-equiv="refresh"> tag in the head section for a smooth transition experience.

Key HTML elements:

  • HTML5 doctype and semantic structure
  • Meta tags for character encoding and viewport settings
  • Container div for centered content
  • Heading elements (h1, h3) for hierarchical content structure
  • Paragraph elements for descriptive text
  • Button-styled anchor tag for navigation

CSS techniques applied:

  • Flexbox for centering and organizing content (display: flex, flex-direction: column)
  • Custom typography with proper font sizing and line height
  • Consistent color scheme matching the health and wellness theme
  • Smooth transitions for interactive elements
  • Responsive design with viewport units and percentage-based sizing
  • Box shadow effects to create depth for the container

Accessibility

  • Semantic HTML structure with proper heading hierarchy
  • Sufficient color contrast between text and background (green/white theme)
  • Descriptive link text for the Page Editor button
  • Meta refresh with adequate time (4 seconds) for users to read the content
  • Emoji use (üçÉ) that complements text content but doesn't replace critical information
  • Responsive design that works across different screen sizes and devices

Volunteer Page

Technical Description

The volunteer page (Volunteer.html) showcases volunteer opportunities and allows users to explore various programs.

Key HTML elements:

  • Semantic sectioning with <header>, <main>, <section>, <footer>
  • Navigation menu with logo and links
  • Card-based layout for volunteer opportunities
  • Form elements for contact information
  • Statistical information display with numerical data
  • Program filtering system using radio buttons
  • Review section with star ratings

CSS techniques applied:

  • CSS Grid for program cards layout (display: grid)
  • Flexbox for navigation and card components
  • Custom styling for form elements with focus states
  • Interactive elements with hover effects
  • Icon integration using image elements
  • Card design with consistent spacing and borders
  • Mobile-responsive layout adjustments
  • Custom button styling with varied visual states

Accessibility

  • Semantic HTML with proper heading hierarchy (h2, h3)
  • Alternative text for all images, including icons and program photos
  • Form labels and required attributes for form validation
  • Proper contrast ratios for text content and backgrounds
  • Keyboard navigable interface for all interactive elements
  • Descriptive button text that indicates purpose
  • Color is not used as the only means of conveying information
  • Emoji supplementing text rather than replacing it
  • Responsive design that adapts to different screen sizes and device capabilities

Content Page

Technical Description

The content page (Content_ST01.html) provides in-depth information about mental health and mindfulness support.

Key HTML elements:

  • Side navigation menu with anchor links to page sections
  • Article structure with multiple content sections
  • Hero image and section images
  • Grid-based content layouts
  • "Back to top" button functionality
  • Consistent header and footer structures

CSS techniques applied:

  • Position: sticky for the side navigation
  • Grid layouts for content organization
  • Custom styling for navigation links with active states
  • Image sizing and responsive adjustments
  • Smooth scrolling behavior for anchor links
  • Typography hierarchy with varied font sizes and weights
  • Spacing and padding for readability
  • Box shadows and border effects for visual hierarchy

Accessibility

  • Semantic HTML5 structure with appropriate landmarks
  • Skip navigation via the side menu for keyboard users
  • Alternative text for all images describing the content
  • Proper heading hierarchy maintaining document outline
  • Sufficient color contrast for all text elements
  • "Go to top" button for easier navigation of long content
  • Responsive images that scale appropriately
  • Focus indicators for keyboard navigation
  • Descriptive link text that explains destination

Challenges and Lessons Learned

  1. Layout Consistency:
    • Challenge: Maintaining a consistent layout across all pages while implementing different functionalities.
    • Lesson: Created reusable CSS classes and components for visual coherence.
  2. Responsive Design:
    • Challenge: Creating layouts that work well on both desktop and mobile devices.
    • Lesson: Used CSS Grid and Flexbox effectively with appropriate media queries.
  3. Filter Functionality:
    • Challenge: Implementing the filter system for volunteer programs.
    • Lesson: Learned about manipulating the DOM with CSS selectors and input states.
  4. Image Management:
    • Challenge: Balancing image quality with page load performance.
    • Lesson: Optimized images and used appropriate sizing techniques.
  5. Form Design:
    • Challenge: Creating an intuitive and accessible contact form.
    • Lesson: Importance of proper labeling, validation, and feedback mechanisms.

Compliance

  1. Content Appropriateness: All content is educational and relates to health and wellness.
  2. Accessibility: Built following WCAG guidelines with proper semantic structure.
  3. Copyright Compliance: All images used are sourced with appropriate licensing.
  4. Privacy Considerations: The contact form includes only necessary information fields.
  5. Performance: Optimized for fast loading with appropriate image sizing.
  6. Security: No client-side scripts contain sensitive information.
  7. Browser Compatibility: Pages work across modern browsers using standard HTML5 and CSS3 features.
